# Сапёр (Minesweeper)

Desktop-приложение «Сапёр» на React + Electron.

Структура проекта описана в [STRUCTURE.md](./STRUCTURE.md).

## Требования

- Node.js 18+
- npm

## Установка

```bash
npm install
```

## Скрипты

| Скрипт | Описание |
|--------|----------|
| `npm run dev` | Запуск Vite dev-сервера (веб) |
| `npm run build` | Сборка React-приложения |
| `npm run electron:dev` | Запуск в Electron (режим разработки) |
| `npm run electron:build` | Сборка Windows .exe |

## Сборка .exe

```bash
npm run electron:build
```

Готовый установщик будет в папке `release/`.

> **Примечание:** Если при сборке возникает ошибка «No JSON content found», попробуйте запустить сборку из пути без кириллицы (например, скопируйте проект в `C:\projects\vibe`).

## Игровой процесс

- **ЛКМ** — открыть клетку
- **ПКМ** — поставить/убрать флаг
- Поле 10×10, 15 мин
- Первая открытая клетка никогда не бывает миной
